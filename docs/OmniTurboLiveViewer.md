




**Important Note for using Viewer with the KaiKai Form component**
Using the live viewer when testing forms **will** cause a lag, there are many details being synced and the viewer is triggered to update every time a change is made to the form. This means that if you are typing in a text field, the viewer will update with each keystroke, which can cause a noticeable lag in the interface (about ~1 second).
This cannot be avoided, Everything will work as expected, but the lag is noticeable and **does not** occur when the viewer is not being used.
